## Tecnologías

- React 19.0.0
- Node.js 14+
- Bootstrap 5.3.5
- React Router 7.3.0
- Axios 1.8.2
- Font Awesome 6.7.2

## Instalación

```bash
npm install
npm start
```

## Uso

1. Accede a la plataforma
2. Explora los planes disponibles
3. Realiza consultas médicas
4. Gestiona tu financiamiento
